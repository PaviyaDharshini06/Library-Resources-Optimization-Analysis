# Library-Resources-Optimization-Analysis
The Library Book Purchase Recommendation System is a Python-based data analytics and machine learning project designed to assist libraries in making intelligent book purchasing decisions. The system analyzes historical borrowing data to identify which books or categories are in high demand and recommends whether the library should purchase additional copies.This project integrates data preprocessing, visualization, natural language processing, and machine learning to generate recommendations. It uses TF-IDF vectorization and cosine similarity to analyze relationships between books based on title, author, and category. A Logistic Regression classification model is then trained to predict whether a book should be purchased.The system also includes an interactive graphical user interface (GUI) built using Tkinter, where users can enter book details and instantly receive purchase recommendations along with confidence scores and visual analytics.
